Static : cas par défaut
Chaque élément :
•  est situé sur le même plan que les autres éléments dans le flux ;
•  se place le plus haut possible et le plus à gauche possible au sein de son parent ;
•  est dépendant de l’élément frère précédent : deux éléments de type block s’empilent verticalement l’un sous l’autre, deux éléments de type inline se suivent sur la même ligne s’ils en ont la place.

Relative
Un élément positionné relativement se place par rapport à sa position classique dans le flux, puis est éventuellement décalé si au moins une des propriétés top, right, bottom ou left est renseignée.

Absolute
un élément positionné en absolu se place par rapport à son premier ancêtre positionné (position: qqchose)

Fixed
l’élé­ment fixé est toujours positionné par rapport à la partie visible de la fenêtre du navigateur

Float : parler du clear. Mentionner l'existence quasi-systématique d'une alternative plus correcte. (Démo par l'image)
